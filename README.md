# 11ty Base Template

A base template for creating static sites with Eleventy and ES modules support.

## Quick Start

```bash
# Install dependencies
npm install

# Build project
npm run build

# Start development server
npm run dev
```

## Structure

```
src/
├── _layouts/     # Templates
├── assets/       # Static files
└── index.md      # Home page
```

## Commands

- `npm run clean` — clean build directory
- `npm run build` — build project
- `npm run dev` — development mode with file watching

## Technologies

- Eleventy 3.1.2
- ES Modules
- Nunjucks
- CSS Grid & Flexbox
